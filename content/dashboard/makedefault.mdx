# Make Default Dashboard

You can make a newly created dashboard to the default. To make the dashboard as default, select **Make Default** from the newly created dashboard drop-down menu.

![makedefaultdashboard](/images/dashboardimages/makedefault.png)

The Dashboard is marked as default.

![makedefaultdashboard](/images/dashboardimages/defaultdashboard.png)

To revert the changes, select **Remove as Default**.

**Note:** You must be authorized to perform this action.
