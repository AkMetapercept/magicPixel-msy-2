# Change Dashboard

You can change the Dashboard that is created and revert the Dashboard display to the Default dashboard.

Perform the following steps to change the Dashboard.

1. From the new Dashboard created, select the **Change Dashboard** option.

2. Click **Project's Default Dashboard** option in the Dashboards screen.

![changedashboard](/images/dashboardimages/changedashboard.png)

**Step Result:** The default dashboard appears on the screen.

Or

3. If you are viewing the Default Dashboard, you can select and navigate to the specific Dashboard to display.
