# Data Elements
The Data Elements are the core components that are to be defined while creation of the tags. Using the Data Elements, you can retrieve data from the website or mobile application through the URL. There are default Data Elements available from the template, so that you can use these default data elements for multiple projects. You may not have to create a new data element every time you create a project if you select **Default Data Elements**. You can create any number of Data elements for a particular Data type and define in the tags as per client requirements.

Some of the Data types include Server Header, Server Cookie, MagicPixel Data Layer, URL Parameter, Custom JavaScript, Custom event, and so on.

**Note**: Using Administrator privileges, you can add new default Data Elements using the Settings option available on the left navigation panel. **Settings -> Templates -> Available Templates**. 

You can navigate to any of the Tag and add new Data Elements from the Data Elements section in the **Apply Template** screen.
